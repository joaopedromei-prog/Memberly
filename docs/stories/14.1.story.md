## Story

**Como** sistema de certificados,
**Quero** uma tabela `certificates` no Supabase com RLS e indexes adequados,
**Para que** o sistema possa registrar e consultar certificados emitidos aos membros que completam 100% de um produto.

## Acceptance Criteria

1. **AC1** — Tabela `certificates` criada com colunas: `id` (UUID PK), `profile_id` (FK → profiles), `product_id` (FK → products), `certificate_url` (text, nullable), `hash` (text, unique), `issued_at` (timestamptz), `created_at` (timestamptz), `updated_at` (timestamptz).
2. **AC2** — Foreign keys com `ON DELETE CASCADE` para `profiles` e `products`.
3. **AC3** — Constraint UNIQUE em `(profile_id, product_id)` — um certificado por membro por produto.
4. **AC4** — Constraint UNIQUE em `hash` — cada certificado tem hash único para validação pública.
5. **AC5** — RLS policies: membros veem apenas seus próprios certificados; admins têm acesso total.
6. **AC6** — Indexes: `idx_certificates_profile` em `profile_id`, `idx_certificates_hash` em `hash`.
7. **AC7** — Coluna `updated_at` com trigger de auto-update (seguindo padrão da migration `20260313000001`).
8. **AC8** — Type definitions atualizadas em `src/types/database.ts` com `certificates` table type.

## Dev Notes

### DDL Reference

```sql
CREATE TABLE certificates (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    profile_id UUID NOT NULL REFERENCES profiles(id) ON DELETE CASCADE,
    product_id UUID NOT NULL REFERENCES products(id) ON DELETE CASCADE,
    certificate_url TEXT,
    hash TEXT NOT NULL UNIQUE,
    issued_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    UNIQUE(profile_id, product_id)
);

CREATE INDEX idx_certificates_profile ON certificates(profile_id);
CREATE INDEX idx_certificates_hash ON certificates(hash);
```

### RLS Pattern (seguir migration 20260311000008)

```sql
ALTER TABLE certificates ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Members can view own certificates"
    ON certificates FOR SELECT
    USING (auth.uid() = profile_id);

CREATE POLICY "Admins have full access to certificates"
    ON certificates FOR ALL
    USING ((auth.jwt() -> 'user_metadata' ->> 'role') = 'admin');
```
