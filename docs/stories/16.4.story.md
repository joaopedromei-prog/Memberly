## Story

**Como** frontend da plataforma,
**Quero** endpoints de API para consultar badges, streaks e disparar avaliacao de badges,
**Para que** as paginas do membro possam exibir conquistas e status de streak em tempo real.

## Acceptance Criteria

1. **AC1** — `GET /api/gamification/badges` retorna lista de todos os badges ativos com campo `unlocked` (boolean) e `unlocked_at` (string | null) indicando se o membro autenticado ja desbloqueou cada badge.
2. **AC2** — `GET /api/gamification/streak` retorna o streak do membro autenticado: `{ current_streak, longest_streak, last_activity_date }`. Se nao existe registro, retorna `{ current_streak: 0, longest_streak: 0, last_activity_date: null }`.
3. **AC3** — `GET /api/gamification/profile` retorna dados combinados: `{ badges: [...], streak: {...} }` para exibicao no perfil do membro.
4. **AC4** — `POST /api/gamification/check` dispara `evaluateBadges(profileId)` para o membro autenticado e retorna `{ unlocked: string[] }` com IDs dos badges recem-desbloqueados.
5. **AC5** — Todos os endpoints usam `authenticateUser()` de `src/lib/utils/auth-guard.ts` para autenticacao.
6. **AC6** — Respostas usam helpers `apiSuccess()` / `apiError()` de `src/lib/utils/api-response.ts`.
7. **AC7** — Testes unitarios para cada endpoint.

### GET /api/gamification/badges

```typescript
import { authenticateUser } from '@/lib/utils/auth-guard';
import { apiSuccess, apiError } from '@/lib/utils/api-response';
import { createServerClient } from '@/lib/supabase/server';

export async function GET() {
  const user = await authenticateUser();
  if (!user) return apiError('Unauthorized', 401);

  const supabase = await createServerClient();

  // Get all active badges
  const { data: badges } = await supabase
    .from('badges')
    .select('*')
    .eq('active', true)
    .order('created_at');

  // Get member's unlocked badges
  const { data: unlocked } = await supabase
    .from('member_badges')
    .select('badge_id, unlocked_at')
    .eq('profile_id', user.id);

  const unlockedMap = new Map(
    unlocked?.map((u) => [u.badge_id, u.unlocked_at]) ?? []
  );

  const result = badges?.map((badge) => ({
    ...badge,
    unlocked: unlockedMap.has(badge.id),
    unlocked_at: unlockedMap.get(badge.id) ?? null,
  })) ?? [];

  return apiSuccess(result);
}
```

### GET /api/gamification/streak

```typescript
export async function GET() {
  const user = await authenticateUser();
  if (!user) return apiError('Unauthorized', 401);

  const supabase = await createServerClient();

  const { data: streak } = await supabase
    .from('streaks')
    .select('current_streak, longest_streak, last_activity_date')
    .eq('profile_id', user.id)
    .maybeSingle();

  return apiSuccess(streak ?? {
    current_streak: 0,
    longest_streak: 0,
    last_activity_date: null,
  });
}
```

### POST /api/gamification/check

```typescript
import { evaluateBadges } from '@/lib/gamification/badge-engine';

export async function POST() {
  const user = await authenticateUser();
  if (!user) return apiError('Unauthorized', 401);

  const unlocked = await evaluateBadges(user.id);

  return apiSuccess({ unlocked });
}
```
