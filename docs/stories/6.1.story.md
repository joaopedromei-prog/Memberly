## Story

**As a** desenvolvedor,
**I want** remover o AI Wizard e toda integração com a Claude API,
**so that** o sistema não dependa mais dessa API e o código fique mais enxuto.

---

## Acceptance Criteria

1. Rota `/admin/products/ai-wizard` removida (página e componentes)
2. Componentes `AiWizardStep1.tsx`, `AiWizardStep2.tsx`, `AiWizardStep3.tsx` removidos de `src/components/admin/ai-wizard/`
3. API route `/api/ai/generate-structure` removida
4. API route `/api/ai/save-structure` removida
5. Arquivo `src/lib/ai/claude-client.ts` removido
6. Arquivo `src/lib/ai/prompts/generate-structure.ts` removido
7. Arquivo `src/lib/ai/save-generated-structure.ts` removido
8. Tipos relacionados ao wizard removidos de `src/types/ai.ts` (manter tipos de banner se existirem)
9. Dependência `@anthropic-ai/sdk` removida do `package.json`
10. Variável `ANTHROPIC_API_KEY` removida do `.env.example`
11. Botão "Criar com IA" removido da página de listagem de produtos (`/admin/products`)
12. Testes relacionados ao wizard e Claude API removidos
13. Build, typecheck e lint passam sem erros após remoção
14. Testes restantes continuam passando
