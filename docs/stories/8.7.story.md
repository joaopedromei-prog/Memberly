## Story

**As a** membro,
**I want** buscar aulas e módulos por nome em todos os meus cursos usando uma busca rápida,
**so that** eu encontre o conteúdo que preciso sem navegar curso por curso.

---

## Context

### Estado Atual

O `MemberHeader.tsx` exibe o nome do usuário e botão de logout. Não existe funcionalidade de busca na área do membro. Para encontrar uma aula específica, o membro precisa navegar produto por produto, módulo por módulo.

### Mudança Necessária

Adicionar ícone de busca (lupa) no header do membro. Ao clicar (ou `Ctrl+K`/`Cmd+K`), abre overlay de busca com input que filtra aulas e módulos em tempo real (debounce 300ms), mostrando resultados apenas de produtos que o membro tem acesso.

### Decisões Técnicas

- Busca server-side via API route (não client-side filtering) — escalável para catálogos grandes
- Query usa `ILIKE` no Supabase com join em `member_access` para filtrar por acesso
- Overlay é um dialog fixo (não modal bloqueante) com foco automático no input
- Resultados navegáveis via keyboard (arrow up/down + Enter)

---

## Acceptance Criteria

1. Ícone de busca (lupa) no `MemberHeader`, ao lado do nome do usuário
2. Ao clicar na lupa, abre search overlay (dialog centralizado) com input de texto com autofocus
3. Busca em tempo real com **debounce 300ms** — chama API a cada mudança de texto
4. Resultados mostram: título da aula, nome do módulo, nome do produto, duração (se disponível)
5. Apenas conteúdo de produtos que o **membro tem acesso** aparece nos resultados
6. Clicar num resultado navega direto para a página da aula (`/products/[slug]/lessons/[id]`)
7. Keyboard shortcut: `Ctrl+K` (Windows/Linux) ou `Cmd+K` (Mac) abre/fecha a busca
8. Keyboard navigation: Arrow Up/Down para navegar resultados, Enter para selecionar, Escape para fechar
9. Estado vazio: "Nenhum resultado encontrado" quando busca retorna vazio
10. Build, typecheck e lint passam

## Dev Notes

### Query de busca

```sql
SELECT
  l.id AS lesson_id,
  l.title AS lesson_title,
  m.title AS module_title,
  p.title AS product_title,
  p.slug AS product_slug,
  l.duration_minutes
FROM lessons l
JOIN modules m ON l.module_id = m.id
JOIN products p ON m.product_id = p.id
JOIN member_access ma ON ma.product_id = p.id
WHERE ma.profile_id = $1 -- authenticated user
  AND (l.title ILIKE $2 OR m.title ILIKE $2)  -- '%search_term%'
  AND l.is_published = true
ORDER BY l.title
LIMIT 20;
```

### Debounce Hook

```typescript
function useDebounce<T>(value: T, delay: number): T {
  const [debouncedValue, setDebouncedValue] = useState(value);
  useEffect(() => {
    const handler = setTimeout(() => setDebouncedValue(value), delay);
    return () => clearTimeout(handler);
  }, [value, delay]);
  return debouncedValue;
}
```

### Keyboard Shortcut

```typescript
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.key === 'k') {
      e.preventDefault();
      setIsSearchOpen(prev => !prev);
    }
  };
  document.addEventListener('keydown', handleKeyDown);
  return () => document.removeEventListener('keydown', handleKeyDown);
}, []);
```
