## Story

**As a** admin,
**I want** editar descrições de aulas com um editor rich text,
**so that** eu possa formatar o conteúdo com negrito, itálico, listas e links.

---

## Acceptance Criteria

1. O formulário de criação/edição de aula substitui o `<textarea>` de descrição por um editor rich text
2. O editor suporta formatação: negrito, itálico, sublinhado, listas (ordenadas e não-ordenadas), links e headings (H2, H3)
3. O conteúdo é salvo em formato HTML no campo `description` da tabela `lessons`
4. A descrição formatada é exibida corretamente na área de membros (player de aula) com sanitização HTML (XSS prevention)
5. O editor rich text também é aplicado nos formulários de descrição de produtos e módulos
6. O editor é responsivo e funciona corretamente em desktop e tablet
7. Toolbar do editor é intuitiva com ícones reconhecíveis para cada ação de formatação
8. O editor permite colar texto de fontes externas com limpeza automática de formatação indesejada (paste cleaning)

## Dev Notes

### Bibliotecas Recomendadas

- **Tiptap** (recomendado): headless, extensível, baseado em ProseMirror, TypeScript-first
- **React Quill**: mais simples, menos customizável
- **DOMPurify**: sanitização HTML contra XSS

### Considerações

- O campo `description` no banco é `TEXT` — já suporta HTML sem alteração de schema
- Sanitizar sempre no lado do client antes de exibir (defense in depth)
- O editor deve ter altura mínima razoável (~200px) e auto-expand
