## Story

**As a** membro,
**I want** marcar aulas como favoritas e ter uma seção de favoritos na home,
**so that** eu possa rever facilmente o conteúdo mais importante para mim.

---

## Context

### Estado Atual

Não existe funcionalidade de favoritos/bookmarks no Memberly. A home do membro exibe "Continue Assistindo" e "Meus Cursos", mas não há uma forma de o membro salvar aulas específicas para acesso rápido.

### Mudança Necessária

1. Criar tabela `lesson_bookmarks` no Supabase
2. Adicionar botão de favoritar na página da aula (toggle coração/estrela)
3. Adicionar seção "Favoritos" na home do membro (carousel)

### Decisões Técnicas

- Tabela simples: `lesson_bookmarks(id, profile_id, lesson_id, created_at)` com UNIQUE em `(profile_id, lesson_id)`
- RLS: membro só vê/gerencia seus próprios bookmarks
- Toggle usa optimistic update (muda visual imediatamente, reverte se API falhar)
- Seção "Favoritos" só aparece se o membro tem ≥1 bookmark

---

## Acceptance Criteria

1. Tabela `lesson_bookmarks` criada com `id` (uuid), `profile_id` (FK profiles), `lesson_id` (FK lessons), `created_at` (timestamptz). UNIQUE constraint em `(profile_id, lesson_id)`
2. RLS: `SELECT`, `INSERT`, `DELETE` somente onde `profile_id = auth.uid()`
3. Botão de favorito (ícone coração) na página da aula, ao lado do título (no `LessonInfo.tsx`)
4. Toggle: coração preenchido = favoritado, coração outline = não favoritado. Optimistic update
5. Seção "Favoritos" na home do membro — carousel horizontal com cards das aulas favoritadas (aparece entre "Continue Assistindo" e "Meus Cursos")
6. Card do favorito mostra: título da aula, nome do módulo, nome do produto, duração. Link direto para a aula
7. Seção "Favoritos" oculta se não houver bookmarks
8. Build, typecheck e lint passam

## Dev Notes

### Toggle API

```typescript
// POST /api/lessons/[id]/bookmark
const existing = await supabase
  .from('lesson_bookmarks')
  .select('id')
  .eq('profile_id', userId)
  .eq('lesson_id', lessonId)
  .single();

if (existing.data) {
  await supabase.from('lesson_bookmarks').delete().eq('id', existing.data.id);
  return { bookmarked: false };
} else {
  await supabase.from('lesson_bookmarks').insert({ profile_id: userId, lesson_id: lessonId });
  return { bookmarked: true };
}
```

### Optimistic Update Pattern

```typescript
const [isBookmarked, setIsBookmarked] = useState(initialState);

async function toggleBookmark() {
  setIsBookmarked(!isBookmarked); // optimistic
  try {
    const res = await fetch(`/api/lessons/${lessonId}/bookmark`, { method: 'POST' });
    const data = await res.json();
    setIsBookmarked(data.bookmarked); // confirm
  } catch {
    setIsBookmarked(isBookmarked); // rollback
    toast.error('Erro ao salvar favorito');
  }
}
```
