## Story

**As an** admin,
**I want** reordenar módulos e aulas via drag & drop de forma consistente, confiável e acessível,
**so that** a organização do curso seja intuitiva e funcione perfeitamente em desktop e dispositivos touch.

---

## Context

### Estado Atual

O componente `SortableList.tsx` em `src/components/ui/` já implementa drag & drop (provavelmente com @dnd-kit). É usado em pelo menos a lista de módulos do admin. Não está confirmado se a lista de aulas também usa, nem se touch/mobile funciona corretamente, nem se há rollback visual em caso de erro da API.

### Mudança Necessária

Auditar a implementação existente de drag & drop em módulos e aulas. Garantir: (1) funciona nos dois níveis, (2) feedback visual adequado, (3) rollback em erro, (4) suporte touch com targets ≥ 44px. Corrigir o que não estiver funcionando.

### Natureza da Story

Esta é uma story de **auditoria + correção**, não de feature nova. O escopo depende do que for encontrado na auditoria.

---

## Acceptance Criteria

1. Drag & drop funciona na lista de **módulos** (`/admin/products/[id]` ou sub-rota de módulos)
2. Drag & drop funciona na lista de **aulas** (`/admin/products/[id]/modules/[moduleId]/lessons`)
3. Feedback visual durante o drag: item levantado com sombra/opacidade, placeholder na posição de destino
4. Ao soltar, API de reorder é chamada e persiste a nova ordem imediatamente
5. Se a API falhar, a lista **reverte** visualmente para a ordem anterior com toast de erro
6. Funciona em dispositivos touch/mobile: drag handle com `touch-action: none`, tamanho mínimo **44×44px**
7. Keyboard accessible: itens podem ser reordenados via teclado (tab + arrow keys ou equivalente)
8. Build, typecheck e lint passam

### Rollback Pattern

```typescript
const handleDragEnd = async (oldIndex: number, newIndex: number) => {
  const previousItems = [...items]; // snapshot
  const reordered = arrayMove(items, oldIndex, newIndex);
  setItems(reordered); // optimistic
  try {
    await reorderAPI(reordered.map((item, i) => ({ id: item.id, sort_order: i })));
  } catch {
    setItems(previousItems); // rollback
    toast.error('Erro ao reordenar. Tente novamente.');
  }
};
```
