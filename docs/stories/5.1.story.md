## Story

**As a** admin,
**I want** informar poucos dados sobre um produto e ter a IA gerando a estrutura completa,
**so that** eu crie áreas de membros em minutos.

---

## Acceptance Criteria

1. Botão "Criar com IA" na página de produtos abre wizard modal/page com steps
2. Step 1 — Inputs: nome do produto, tema/nicho, público-alvo (texto livre), número de módulos desejado (slider 1-20), tom do conteúdo (formal/informal)
3. Step 2 — Geração: loading state com mensagem "Gerando sua área de membros..." (chamada à Claude API via Route Handler)
4. Claude API recebe prompt estruturado com os inputs e retorna JSON com: estrutura de módulos (título, descrição), aulas por módulo (título, descrição), sugestões de banners (texto descritivo para geração posterior ou referência)
5. Step 3 — Preview: exibir a estrutura gerada em formato visual (árvore de módulos e aulas) para revisão
6. Botão "Aprovar e Criar" salva toda a estrutura no banco (produto + módulos + aulas) de uma vez
7. Botão "Regenerar" permite nova geração com os mesmos ou novos inputs
8. Tratamento de erro: se a API falhar, exibir mensagem amigável com opção de retry

## Dev Notes

### Claude API Integration [Source: architecture.md#6-external-apis]

```typescript
// src/lib/ai/claude-client.ts
import Anthropic from '@anthropic-ai/sdk';

const anthropic = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY!,
});

export async function generateStructure(inputs: WizardInputs): Promise<GeneratedStructure> {
  const message = await anthropic.messages.create({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 4096,
    messages: [
      {
        role: 'user',
        content: buildStructurePrompt(inputs),
      },
    ],
  });

  const textBlock = message.content.find((block) => block.type === 'text');
  if (!textBlock || textBlock.type !== 'text') {
    throw new Error('No text response from Claude API');
  }

  return parseAndValidateStructure(textBlock.text);
}
```

### Prompt Engineering Notes

O prompt deve:
1. Instruir output estritamente em JSON (sem markdown, sem explicações)
2. Especificar o schema exato esperado
3. Incluir contexto do nicho, público-alvo e tom
4. Pedir número exato de módulos conforme slider
5. Pedir 3-7 aulas por módulo (variável, realista)
6. Pedir duração estimada em minutos para cada aula
7. Pedir `bannerSuggestion` como texto descritivo (para geração de imagem futura)

```typescript
// src/lib/ai/prompts/generate-structure.ts
export function buildStructurePrompt(inputs: WizardInputs): string {
  return `Você é um especialista em criação de cursos online.
Gere a estrutura completa de um curso com base nos seguintes dados:

- Nome do Produto: ${inputs.productName}
- Tema/Nicho: ${inputs.topic}
- Público-alvo: ${inputs.targetAudience}
- Número de Módulos: ${inputs.moduleCount}
- Tom: ${inputs.tone === 'formal' ? 'Formal e profissional' : 'Informal e acessível'}

Retorne APENAS um JSON válido (sem markdown, sem explicações) com o seguinte schema:
{
  "product": {
    "title": "string",
    "description": "string (2-3 frases)",
    "bannerSuggestion": "string (descrição visual para gerar banner)"
  },
  "modules": [
    {
      "title": "string",
      "description": "string (1-2 frases)",
      "bannerSuggestion": "string",
      "lessons": [
        {
          "title": "string",
          "description": "string (1 frase)",
          "durationMinutes": number (5-60)
        }
      ]
    }
  ]
}

Regras:
- Exatamente ${inputs.moduleCount} módulos
- 3-7 aulas por módulo (varie conforme complexidade)
- Títulos claros e atrativos para o público-alvo
- Descrições no tom ${inputs.tone}
- Durações realistas (5-60 minutos)
- bannerSuggestion deve descrever uma imagem 16:9 adequada ao tema`;
}
```

### Error Handling for AI [Source: architecture.md#19-checklist-results]

> Circuit breaker para AI APIs: Claude/Gemini não possuem fallback circuit breaker. Implementar timeout + retry com backoff exponencial + mensagem graceful no AI wizard.

```typescript
// Retry pattern
const MAX_RETRIES = 2;
const TIMEOUT_MS = 30000;

for (let attempt = 0; attempt <= MAX_RETRIES; attempt++) {
  try {
    const result = await Promise.race([
      generateWithClaude(inputs),
      timeout(TIMEOUT_MS),
    ]);
    return result;
  } catch (error) {
    if (attempt === MAX_RETRIES) throw error;
    if (error.status === 429) {
      await sleep(Math.pow(2, attempt) * 1000); // Exponential backoff
    }
  }
}
```
