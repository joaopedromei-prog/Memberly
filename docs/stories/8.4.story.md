## Story

**As an** admin,
**I want** duplicar um produto inteiro incluindo todos os seus módulos e aulas,
**so that** eu possa criar variações de cursos sem precisar recriar tudo do zero.

---

## Context

### Estado Atual

O admin pode criar produtos, módulos e aulas individualmente. Não existe funcionalidade de duplicação. Para criar uma variação de um curso existente, o admin precisa recriar manualmente cada módulo e aula.

### Mudança Necessária

Adicionar botão "Duplicar" no menu de ações do produto (admin). A duplicação faz deep copy: produto → módulos → aulas, gerando um novo produto em rascunho com slug único. Product_mappings NÃO são copiados para evitar conflitos de webhook.

### Decisões Técnicas

- A duplicação é uma operação atômica via Supabase (usa supabaseAdmin para inserir tudo numa sequência)
- Slug do produto duplicado: `{slug-original}-copy` ou `{slug-original}-copy-2` se já existir
- Banner URLs são copiadas por referência (não duplica arquivos no Storage)
- Attachments de aulas também copiados por referência

---

## Acceptance Criteria

1. Botão "Duplicar Produto" acessível no menu de ações do produto (action menu ou página de edição)
2. Ao clicar, abre dialog de confirmação com nome sugerido "Cópia de {título original}" (editável)
3. Deep copy: produto + todos módulos + todas aulas, preservando sort_order, vídeos, attachments, descriptions
4. Produto duplicado nasce como `is_published: false` (rascunho)
5. Slug gerado automaticamente: `{slug-original}-copy` (incrementa `-copy-N` se já existir)
6. `product_mappings` NÃO são duplicados (evita conflito de webhook)
7. Após sucesso, redireciona para a página de edição do produto duplicado
8. Toast de sucesso: "Produto duplicado com sucesso"
9. Build, typecheck e lint passam

## Dev Notes

### Fluxo de duplicação

```
POST /api/products/[id]/duplicate { title: "Cópia de Curso X" }

1. Busca produto original com módulos e aulas
2. Gera slug único
3. INSERT produto (is_published: false)
4. Para cada módulo:
   a. INSERT módulo com novo UUID, product_id = novo produto
   b. Para cada aula do módulo:
      INSERT aula com novo UUID, module_id = novo módulo
5. Retorna { id, slug } do novo produto
```

### Slug único

```typescript
async function generateUniqueSlug(baseSlug: string): Promise<string> {
  let slug = `${baseSlug}-copy`;
  let counter = 1;
  while (await slugExists(slug)) {
    counter++;
    slug = `${baseSlug}-copy-${counter}`;
  }
  return slug;
}
```
