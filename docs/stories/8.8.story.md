## Story

**As a** membro,
**I want** ver exatamente de onde parei em cada curso (nome da aula, módulo, e quando assisti por último),
**so that** eu possa retomar meus estudos sem perder tempo procurando onde estava.

---

## Context

### Estado Atual

A seção "Continue Assistindo" na home do membro (`app/(member)/page.tsx`) mostra produtos com progresso entre 0% e 100%. O `ContinueWatchingCard.tsx` já existe mas exibe apenas o nome do produto e progresso geral — não mostra qual aula foi assistida por último nem contexto do módulo.

A tabela `lesson_progress` tem colunas `completed` e `completed_at`, mas não registra quando o membro simplesmente acessou/assistiu uma aula sem completar.

### Mudança Necessária

1. Adicionar coluna `last_watched_at` em `lesson_progress` — atualizada toda vez que o membro acessa uma aula (não só ao completar)
2. Refatorar o `ContinueWatchingCard` para mostrar: nome da última aula, módulo, tempo relativo ("há 2h"), e link direto para continuar ou para a próxima aula não concluída

---

## Acceptance Criteria

1. Coluna `last_watched_at` (timestamptz) adicionada em `lesson_progress` — default `now()`
2. Toda vez que o membro acessa uma aula, `last_watched_at` é atualizado (upsert em lesson_progress)
3. Seção "Continue Assistindo" mostra cards com: **nome do produto**, **última aula assistida** (título), **módulo** da aula, **progresso geral** (%), e **tempo relativo** ("há 2 horas", "ontem", "há 3 dias")
4. Se o membro **não completou** a última aula, card mostra "Continue: {aula}" com link direto
5. Se **completou**, card mostra "Próxima: {próxima aula não concluída}" com link direto
6. Cards ordenados por `last_watched_at` mais recente primeiro
7. Tempo relativo usa `Intl.RelativeTimeFormat` (sem lib externa)
8. Build, typecheck e lint passam

## Dev Notes

### Relative Time Helper

```typescript
// src/lib/utils/relative-time.ts
export function getRelativeTime(dateStr: string): string {
  const rtf = new Intl.RelativeTimeFormat('pt-BR', { numeric: 'auto' });
  const diff = Date.now() - new Date(dateStr).getTime();
  const minutes = Math.floor(diff / 60_000);
  if (minutes < 60) return rtf.format(-minutes, 'minute');
  const hours = Math.floor(minutes / 60);
  if (hours < 24) return rtf.format(-hours, 'hour');
  const days = Math.floor(hours / 24);
  if (days < 7) return rtf.format(-days, 'day');
  const weeks = Math.floor(days / 7);
  return rtf.format(-weeks, 'week');
}
```

### Watch API (upsert)

```sql
INSERT INTO lesson_progress (profile_id, lesson_id, last_watched_at)
VALUES ($1, $2, now())
ON CONFLICT (profile_id, lesson_id)
DO UPDATE SET last_watched_at = now();
```
