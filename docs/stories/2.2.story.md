## Story

**As a** admin,
**I want** criar, editar, reordenar e excluir módulos dentro de um produto,
**so that** eu organize o conteúdo em seções lógicas.

---

## Acceptance Criteria

1. Página `/admin/products/[id]/modules` lista módulos do produto com: título, banner, número de aulas, ordem
2. Formulário de criação/edição de módulo: título, descrição, banner (upload)
3. Drag-and-drop para reordenar módulos (atualiza `sort_order` no banco)
4. Exclusão de módulo com confirmação (alerta se contém aulas)
5. Breadcrumb: Produtos > [Nome do Produto] > Módulos
6. Módulos são exibidos na ordem definida pelo `sort_order`

### Database Schema [Source: architecture.md#8-database-schema]

```sql
CREATE TABLE modules (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    product_id UUID NOT NULL REFERENCES products(id) ON DELETE CASCADE,
    title TEXT NOT NULL,
    description TEXT NOT NULL DEFAULT '',
    banner_url TEXT,
    sort_order INTEGER NOT NULL DEFAULT 0,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_modules_product ON modules(product_id);
```

### Supabase Query Patterns

**List modules with lesson count:**
```typescript
const { data, error } = await supabase
  .from('modules')
  .select('*, lessons(count)')
  .eq('product_id', productId)
  .order('sort_order');
```

**Batch reorder:**
```typescript
// Receive: [{ id: 'uuid1', sort_order: 0 }, { id: 'uuid2', sort_order: 1 }, ...]
for (const item of items) {
  await supabase
    .from('modules')
    .update({ sort_order: item.sort_order })
    .eq('id', item.id);
}
```

**Delete with cascade (automatic via FK ON DELETE CASCADE):**
```typescript
const { error } = await supabase
  .from('modules')
  .delete()
  .eq('id', moduleId);
// Lessons are automatically deleted via CASCADE
```

**Get lesson count before delete (for warning):**
```typescript
const { count } = await supabase
  .from('lessons')
  .select('*', { count: 'exact', head: true })
  .eq('module_id', moduleId);
```

### Route Handler Template — Modules

```typescript
// src/app/api/products/[id]/modules/route.ts
import { createServerSupabaseClient } from '@/lib/supabase/server';
import { apiError, apiSuccess } from '@/lib/utils/api-response';
import { type NextRequest } from 'next/server';

export async function GET(
  request: NextRequest,
  { params }: { params: Promise<{ id: string }> }
) {
  const { id: productId } = await params;
  const supabase = await createServerSupabaseClient();

  const { data, error } = await supabase
    .from('modules')
    .select('*, lessons(count)')
    .eq('product_id', productId)
    .order('sort_order');

  if (error) return apiError('FETCH_ERROR', error.message, 500);
  return apiSuccess(data);
}

export async function POST(
  request: NextRequest,
  { params }: { params: Promise<{ id: string }> }
) {
  const { id: productId } = await params;
  const supabase = await createServerSupabaseClient();
  const body = await request.json();

  if (!body.title) {
    return apiError('VALIDATION_ERROR', 'Title is required', 400);
  }

  // Get next sort_order
  const { data: existing } = await supabase
    .from('modules')
    .select('sort_order')
    .eq('product_id', productId)
    .order('sort_order', { ascending: false })
    .limit(1);

  const nextOrder = existing && existing.length > 0 ? existing[0].sort_order + 1 : 0;

  const { data, error } = await supabase
    .from('modules')
    .insert({ ...body, product_id: productId, sort_order: nextOrder })
    .select()
    .single();

  if (error) return apiError('CREATE_ERROR', error.message, 500);
  return apiSuccess(data, 201);
}
```
