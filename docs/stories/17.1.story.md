## Story

**Como** plataforma Memberly,
**Quero** ter um servico de email integrado via Resend,
**Para que** o sistema possa enviar emails transacionais (boas-vindas, magic links, etc.) de forma confiavel.

## Acceptance Criteria

1. **AC1** ‚Äî Pacote `resend` instalado como dependencia do projeto (`packages/memberly-app/`).
2. **AC2** ‚Äî Cliente Resend configurado em `src/lib/email/resend-client.ts` usando variavel de ambiente `RESEND_API_KEY`.
3. **AC3** ‚Äî Funcao utilitaria `sendEmail({ to, subject, html })` exportada de `src/lib/email/send-email.ts` que encapsula o envio via Resend com tratamento de erros e logging.
4. **AC4** ‚Äî Funcao `sendWelcomeEmail({ memberName, memberEmail, productName, loginUrl })` exportada de `src/lib/email/templates/welcome-email.ts` que monta o HTML do email de boas-vindas usando os templates ja configurados em `site_settings` (keys: `welcome_email_subject`, `welcome_email_body`, `welcome_email_active`).
5. **AC5** ‚Äî Se `welcome_email_active` estiver `false` em `site_settings`, a funcao `sendWelcomeEmail` retorna sem enviar (no-op).
6. **AC6** ‚Äî Variavel `RESEND_API_KEY` adicionada ao `.env.example` com comentario descritivo.
7. **AC7** ‚Äî Variavel `EMAIL_FROM` adicionada ao `.env.example` (default: `noreply@memberly.app`) para configurar o remetente.
8. **AC8** ‚Äî Testes unitarios para `sendEmail` e `sendWelcomeEmail` com mock do cliente Resend (minimo 4 test cases: envio ok, API error, template inativo, variavel ausente).

## Dev Notes

### Contexto existente

O admin panel ja possui UI de configuracao de email (`EmailSettings.tsx`) que salva templates em `site_settings`:
- `welcome_email_active` (boolean)
- `welcome_email_subject` (string com `{{product_name}}`)
- `welcome_email_body` (string com `{{member_name}}`, `{{product_name}}`, `{{login_url}}`, `{{platform_name}}`)

A infraestrutura de templates ja esta pronta ‚Äî falta apenas o servico de envio.

### Resend API

```typescript
import { Resend } from 'resend';
const resend = new Resend(process.env.RESEND_API_KEY);
await resend.emails.send({ from, to, subject, html });
```
