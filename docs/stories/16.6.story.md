## Story

**Como** membro da plataforma,
**Quero** ver meu streak de dias consecutivos de estudo no dashboard,
**Para que** eu me sinta motivado a manter a consistencia e nao quebrar minha sequencia.

### StreakCounter Component Pattern

```typescript
// src/components/member/StreakCounter.tsx
'use client';

import { useEffect, useState } from 'react';
import { motion } from 'motion/react';

interface StreakData {
  current_streak: number;
  longest_streak: number;
  last_activity_date: string | null;
}

const MILESTONES = [7, 30];

export function StreakCounter() {
  const [streak, setStreak] = useState<StreakData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch('/api/gamification/streak')
      .then((res) => res.json())
      .then((data) => setStreak(data.data))
      .finally(() => setLoading(false));
  }, []);

  if (loading) return <StreakCounterSkeleton />;
  if (!streak) return null;

  const isMilestone = MILESTONES.includes(streak.current_streak);

  return (
    <div className="rounded-xl border border-dark-border bg-dark-card p-6">
      <div className="flex items-center gap-4">
        <motion.div
          className="text-4xl"
          animate={isMilestone ? {
            scale: [1, 1.2, 1],
            filter: ['brightness(1)', 'brightness(1.5)', 'brightness(1)'],
          } : {}}
          transition={{ duration: 1, repeat: Infinity, repeatDelay: 2 }}
        >
          🔥
        </motion.div>
        <div>
          {streak.current_streak > 0 ? (
            <>
              <p className="text-3xl font-bold text-white">
                {streak.current_streak}
              </p>
              <p className="text-sm text-gray-400">dias consecutivos</p>
            </>
          ) : (
            <p className="text-sm text-gray-400">
              Comece sua sequencia hoje!
            </p>
          )}
        </div>
      </div>
      {streak.longest_streak > 0 && (
        <p className="mt-3 text-xs text-gray-500">
          Recorde: {streak.longest_streak} dias
        </p>
      )}
    </div>
  );
}
```

### Animation Notes

- Usar `motion/react` (NAO `framer-motion`) conforme padrao do projeto
- Milestone animation: pulse + glow no icone de fogo quando current_streak e 7 ou 30
- Animacao sutil, nao invasiva — loop lento com repeatDelay

### Design System References

- Card: `bg-dark-card` com `border-dark-border` e `rounded-xl`
- Numero grande: `text-3xl font-bold text-white`
- Labels: `text-sm text-gray-400`
- Info secundaria: `text-xs text-gray-500`

### Edge Cases

- **Streak = 0:** Exibir mensagem motivacional, sem numero (AC5)
- **Sem registro de streak:** API retorna defaults (0, 0, null) — tratar como streak = 0
- **Streak muito alto:** Numeros de 3+ digitos devem caber no layout
- **Longest streak = 0:** Nao exibir linha de recorde
