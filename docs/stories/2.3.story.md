## Story

**As a** admin,
**I want** criar aulas com vídeos do YouTube ou Panda Video e anexar PDFs,
**so that** o conteúdo esteja completo para os membros.

---

## Acceptance Criteria

1. Página `/admin/products/[id]/modules/[moduleId]/lessons` lista aulas do módulo com: título, provider de vídeo, duração, ordem
2. Formulário de criação/edição de aula: título, descrição, provider de vídeo (select: YouTube / Panda Video), ID ou URL do vídeo, duração estimada (minutos), upload de PDF (opcional)
3. Preview do vídeo embed no formulário de edição (exibe o player embedado conforme o provider selecionado)
4. PDF uploaded para Supabase Storage com URL salva na aula
5. Drag-and-drop para reordenar aulas dentro do módulo
6. Exclusão de aula com confirmação
7. Breadcrumb: Produtos > [Produto] > Módulos > [Módulo] > Aulas

### Video Embed Patterns [Source: architecture.md#16-coding-standards]

**CRITICAL:** Never embed video directly from URL. Always use the `VideoPlayer` component which sanitizes and validates the provider.

**YouTube embed URL:**
```typescript
function getYouTubeEmbedUrl(videoId: string): string {
  return `https://www.youtube.com/embed/${videoId}?rel=0&modestbranding=1`;
}
```

**Panda Video embed URL:**
```typescript
function getPandaVideoEmbedUrl(videoId: string): string {
  return `https://player-vz-${videoId.substring(0, 8)}.tv.pandavideo.com.br/embed/?v=${videoId}`;
}
```

**YouTube ID extraction (common URL formats):**
```typescript
// https://www.youtube.com/watch?v=VIDEO_ID
// https://youtu.be/VIDEO_ID
// https://www.youtube.com/embed/VIDEO_ID
// Direct ID: VIDEO_ID (11 chars)
const YOUTUBE_REGEX = /(?:youtube\.com\/(?:watch\?v=|embed\/)|youtu\.be\/)([a-zA-Z0-9_-]{11})/;
```

**VideoPlayer component pattern:**
```typescript
// src/components/shared/VideoPlayer.tsx
'use client';

interface VideoPlayerProps {
  provider: 'youtube' | 'pandavideo';
  videoId: string;
  className?: string;
}

export function VideoPlayer({ provider, videoId, className }: VideoPlayerProps) {
  const embedUrl = provider === 'youtube'
    ? `https://www.youtube.com/embed/${videoId}?rel=0&modestbranding=1`
    : `https://player-vz-${videoId.substring(0, 8)}.tv.pandavideo.com.br/embed/?v=${videoId}`;

  return (
    <div className={cn('aspect-video w-full', className)}>
      <iframe
        src={embedUrl}
        className="h-full w-full rounded-lg"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
        allowFullScreen
      />
    </div>
  );
}
```

### Supabase Query Patterns

**List lessons with sort_order:**
```typescript
const { data, error } = await supabase
  .from('lessons')
  .select('*')
  .eq('module_id', moduleId)
  .order('sort_order');
```

**Create lesson with auto sort_order:**
```typescript
const { data: existing } = await supabase
  .from('lessons')
  .select('sort_order')
  .eq('module_id', moduleId)
  .order('sort_order', { ascending: false })
  .limit(1);

const nextOrder = existing && existing.length > 0 ? existing[0].sort_order + 1 : 0;

const { data, error } = await supabase
  .from('lessons')
  .insert({
    module_id: moduleId,
    title: body.title,
    description: body.description || '',
    video_provider: body.video_provider,
    video_id: body.video_id,
    pdf_url: body.pdf_url || null,
    sort_order: nextOrder,
    duration_minutes: body.duration_minutes || null,
  })
  .select()
  .single();
```
