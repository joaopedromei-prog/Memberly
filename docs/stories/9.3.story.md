## Story

**As a** membro acessando pelo celular,
**I want** que a seção "Meus Cursos" tenha margens laterais adequadas,
**so that** os cards de produtos não colem nas bordas da tela, mantendo o visual profissional.

---

## Context

### Estado Atual

No mobile (testado em iPhone 14 Pro Max — 430px), a seção "Meus Cursos" no dashboard do membro tem os cards colando na borda direita da tela. O componente `Carousel.tsx` usa `px-4` no container scrollável, porém containers com `overflow-x-auto` não respeitam o `padding-right` em CSS — é um bug conhecido do CSS onde o padding no final do eixo de scroll é ignorado pelo browser.

### Mudança Necessária

Corrigir o `Carousel.tsx` para que o padding direito seja respeitado mesmo em containers com overflow horizontal. A solução padrão é adicionar um pseudo-elemento ou spacer no final do scroll container para forçar o espaçamento.

### Decisões Técnicas

- Usar pseudo-elemento `after` com Tailwind para criar um spacer invisível no final do scroll
- Alternativa: aplicar padding via inner wrapper ao invés de diretamente no container de overflow
- Manter os mesmos breakpoints responsivos existentes (`px-4 sm:px-6 lg:px-16`)

## Dev Notes

### Problema Técnico (CSS Overflow Padding)

Containers com `overflow-x-auto` em CSS não respeitam `padding-right` (ou `padding-inline-end`). O padding é aplicado ao container, mas o browser ignora o padding no final do eixo de scroll. Isso faz com que o conteúdo cole na borda direita.

**Solução recomendada:** Adicionar um pseudo-elemento `after` com largura mínima ao container flex para criar um spacer invisível no final:
```css
/* Exemplo com Tailwind */
after:content-[''] after:min-w-[16px] after:flex-shrink-0
```

Ou usar `scroll-padding-inline` no container scrollável.

### Código Atual (Carousel.tsx)

```tsx
// Linha 64 - Container scrollável
className="scrollbar-hide flex gap-3 overflow-x-auto snap-x snap-mandatory px-4 sm:px-6 lg:px-16 pb-8 pt-4 -mt-4"
```

O `px-4` deveria dar 16px nas laterais, mas o padding-right é ignorado pelo overflow.
